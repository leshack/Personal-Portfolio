Cybersecurity and Ethical Hacking

Senselearner Technologies Pvt. Ltd.

  Sep 2023 - Nov 2023 | 3 mos

I was given the mandate by Senselearner Technologies Pvt Ltd to perform a comprehensive reconnaissance on their systems. This task required meticulous attention to detail and a deep understanding of various tools and techniques. During this engagement,I leveraged various specialized tools and resources. These included:

  • Web Analysis Tools: BuiltWith to identify all technologies used by the website.
  • Network Scanners: Shodan for discovering devices connected to the internet, and Masscan for high-speed IP address scanning.
  • Domain and IP Tools: Who.is, BGP.he.net, and ARIN Whois for detailed domain and IP address information.
  • Subdomain Enumeration: Tools like Sudominizer, OneForAll, and Hakrawler to identify subdomains.
  • Vulnerability Scanners: Nuclei for customizable vulnerability scanning, and Brutespray for automated brute force attacks on network services.
  • Metadata Extraction: Exiftool for extracting metadata from various file types.

Next, I conducted a thorough footprinting exercise using tools such as WordPress Plugins (Yoast Plugins, WPBakery, Elementor), Google Fonts, and CDN services (Cloudflare JS, BootstrapCDN). This mapped the infrastructure, from web servers (Apache, nginx) and hosting providers (Google Cloud, Netmagic Solutions) to JavaScript libraries (jQuery, Vue, GSAP) and CMS (WordPress versions).

Identifying outdated software was critical part of this process. I discovered older versions of WordPress and JavaScript libraries, posing significant security risks, such as WordPress 4.9 and jQuery 2.2.3, known for vulnerabilities.

My detailed reconnaissance identified critical vulnerabilities within Senselearner’s infrastructure. These were comprehensively documented in my report, providing actionable remediation insights.
